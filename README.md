# dbmsQuest 📚💻

dbmsQuest is a premium, client-side interactive learning platform designed to help students master SQL and database management. Powered by client-side SQLite (via `sql.js` WebAssembly), it compiles and tests queries with zero latency.

---

## 🌟 Key Features

* **Interactive SQLite Sandbox**: Run and verify queries against seed schemas directly in the browser.
* **Streamlined Curriculum**: 33 chapters structured logically from Basics to Intermediate, Advanced Projects, and Practice Sets.
* **Lecture Notes Reader**: Built-in document deck to read educational materials.
* **Adjustable Interface**: Drag-to-resize workspace columns and resizable sidebar width.
* **Privacy & Persistence**: Entirely local, saving progress automatically to `localStorage`.

---

## 🚀 Getting Started

### Installation

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

### Production Build

```bash
# Build the application
npm run build

# Start production server
npm run start
```
